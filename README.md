# 🍽️ Recipe Management Web App

A full-stack Django web application that allows users to create, view, update, delete, and manage their personal recipes. The platform also includes a **wishlist feature** to bookmark favorite recipes. Designed with a modern, responsive UI using Tailwind CSS.

---

## 🚀 Features

- ✅ User authentication (login/logout/register)
- 📝 Add, update, and delete your own recipes
- 🌟 Wishlist: Save your favorite recipes to revisit later
- 🖼️ Image upload for each recipe
- 🔐 Access control: Only authors can edit or delete their recipes
- 📱 Fully responsive interface with Tailwind CSS

---

## 🛠️ Tech Stack

- **Backend:** Django
- **Frontend:** HTML, Tailwind CSS
- **Database:** SQLite
- **Authentication:** Django's built-in auth system

## 🧑‍💻 Setup Instructions

 **Clone the repository**
   ```bash
   https://github.com/codesayan2004/recipe_proj.git
   cd recipe_proj
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   python3 manage.py makemigrations
   python3 manage.py migrate
   python manage.py runserver
   ```
